Audio File Instructions

How to add and manage audio files

File Requirements

  • Files must be MP3 format
  • Place files in /audio_files/ folder on SD card
  • Name files as 0001.mp3, 0002.mp3, etc.
  • Maximum 255 files supported
  • Recommended: 8-32 kHz, mono, 64-128 kbps
